# MedBe Pharmacy Website

Official website for **MedBe Pharmacy**, designed to showcase the pharmacy’s services, vision, mission, and community trust while providing a modern and accessible digital presence.

The website highlights MedBe's philosophy of **“Care Beyond Medicine”**, focusing on accessible healthcare, patient guidance, and community wellbeing.

## Project Overview

This website was developed as a **brand presence platform** for MedBe Pharmacy.
The goal is to provide visitors with clear information about the pharmacy, its services, values, and commitment to accessible healthcare.

The platform currently serves as an **informational website**, with future plans to integrate a **full online pharmacy store** once API access and backend services become available.

---

## Key Features

* Modern healthcare-themed UI with MedBe branding
* Responsive design optimized for desktop, tablet, and mobile
* Vision and mission presentation
* Service highlights and pharmacy offerings
* Customer trust section with real Google reviews
* Community-focused pharmacy values
* Store page placeholder for future e-commerce integration
* Smooth UI animations and interactive cards
